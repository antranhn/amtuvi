# 伯 Bác c.
**伯 Bác** *c.*  
Anh cha, vai lớn hơn cha.

- — ruột. Anh ruột cha.

- — họ. Vai anh cha mà cách đời ; đường bá.

- Cô —. Vai chị vai anh cha ; kẻ đáng cho mình kêu bằng chị bằng anh cha, về vai lớn hơn.

- Chú —. Vai em, vai anh cha, đáng đạo anh, đạo em cha mình.
